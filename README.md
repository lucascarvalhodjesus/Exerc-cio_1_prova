#  Exercício 1 - Prova

## Sobre o projeto
Este repositório contém a resolução de um exercício proposto em avaliação acadêmica, com o objetivo de praticar conceitos fundamentais de programação e lógica.

O projeto foi desenvolvido com foco em:
- Organização de código
- Aplicação de conceitos básicos
- Clareza e legibilidade

---

## Tecnologias utilizadas
- Java
- Paradigma de Programação Orientada a Objetos (POO)

---

## Estrutura do projeto
O projeto está organizado da seguinte forma:

```
src/
├── (classes principais)
├── (classes auxiliares, se houver)
```
